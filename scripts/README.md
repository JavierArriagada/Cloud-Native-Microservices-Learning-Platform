# Scripts

Scripts de utilidad para desarrollo, testing y operaciones.

## Scripts planeados

- **seed_data.py** - Carga de datos de ejemplo
- **backup.sh** - Backup de base de datos
- **deploy.sh** - Scripts de deployment
- **healthcheck.sh** - Verificación de salud de servicios
- **init_db.py** - Inicialización de base de datos

## Uso

Los scripts se pueden ejecutar directamente o a través de comandos Make.
